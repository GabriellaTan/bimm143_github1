> set of input protein structures?

``` r
graph.cA.aB <- function(x) {
  x.chainA <- trim.pdb(x, chain="A", elety="CA")
  x.b <- x.chainA$atom$b
  plotb3(x.b, sse=x.chainA, typ="l", ylab="Bfactor")
}
```

according to what is inputted into the function.
